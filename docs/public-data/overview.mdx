Eden Public Data is a collection of public datasets stored in BigQuery and collected using a variety of ETL processes, described in the documentation of each dataset.

## Prerequisites

- [BigQuery account &nearr;](https://cloud.google.com/bigquery/)

## Public Datasets

### MEV-Boost

  - [MEV-Boost Bids](/public-data/mev-boost/bids) - builder bids received, collected from relays of MEV-Boost ecosystem
  - [MEV-Boost Payloads](/public-data/mev-boost/payloads) - payloads delivered, collected from relays of MEV-Boost ecosystem

:::info Join the conversation

See our post on the [Flashbots forum &nearr;](https://collective.flashbots.net/t/mev-boost-public-data-relay-bids-and-payloads/2734)

:::

### Flashbots

  - [Mempool Dumpster](/public-data/flashbots/mempool-dumpster) - transactions, detected from Flashbots Mempool Dumpster
  - [MEV-Share](/public-data/flashbots/mev-share) - transactions, detected from Flashbots MEV-Share

### Gnosis

  - [MEV Blocker](/public-data/gnosis/mev-blocker) - transactions, detected from Gnosis MEV Blocker

### Ethereum Auxiliary

  - [Tags by Pubkey](/public-data/ethereum-auxiliary/tags-pubkey) - tags by pubkey for Ethereum

Eden Public Data is a collection of public datasets stored in BigQuery and collected using a variety of ETL processes, described in the documentation of each dataset.

## Prerequisites

- BigQuery account

## Public Datasets

### MEV-Boost

  - [MEV-Boost Bids](./mev-boost/bids) - builder bids received, collected from relays of MEV-Boost ecosystem
  - [MEV-Boost Payloads](./mev-boost/payloads) - payloads delivered, collected from relays of MEV-Boost ecosystem

### Flashbots

  - [Mempool Dumpster](./flashbots/mempool-dumpster) - transactions, detected from Flashbots Mempool Dumpster
  - [MEV-Share](./flashbots/mev-share) - transactions, detected from Flashbots MEV-Share

### Gnosis

  - [MEV Blocker](./gnosis/mev-blocker) - transactions, detected from Gnosis MEV Blocker
